# Dataset Info.

- **train.csv**  
    id : 샘플 고유 ID  
    context : 금융 및 경제 뉴스 기사 관련 정보  
    question : context 정보 기반 질문  
    answer : 질문에 대한 정답 (Target)  
    33716개 샘플
    
- **test.csv**  
    id : 샘플 고유 ID  
    context : 금융 및 경제 뉴스 기사 관련 정보  
    question : context 정보 기반 질문  
    1507개 샘플
